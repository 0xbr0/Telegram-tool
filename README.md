# Telegram-tool
THIS TOOL ALLOWS YOU TO ADD MEMBERS TO YOUR TELEGRAM CHANNELS

```
Installation termux

$ apt update && apt upgrade
$ pkg install git
$ git clone https://www.github.com/g1ng3rb1t3/Telegram-tool
$ cd Telegram-tool
$ chmod +x install.sh
$ ./install.sh
$ python3 scrapper.py
$ python3 adder.py
```
